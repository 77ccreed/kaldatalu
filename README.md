# Kalda talu homepage

The project is Kalda talu homepage.

## Setup Instructions

* Download repository (ZIP file) or git clone https://github.com/77ccreed/kaldatalu 
* Then open index.html

### Code Dependencies

The website is built with the following dependencies:

* Google Fonts ( Cinzel )
* Google Maps
* Google Analytics
* Font Awesome icons
* Bootstrap 4.0.0
* jQuery 3.2.1
* Images

### Built With

* HTML5
* CSS3
* JavaScript
